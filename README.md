# Fastify Template

My fastify template with mongodb

# Installation

```
degit qxb3/fastify-template my-awesome-api
cd my-awesome-api
npm install
```

# Running

```
npm run start # Running prod
npm run dev # Running dev
npm run lint # Lint code using eslint
```
